About this page

This page is a demonstration of NLI-GO, a natural language library, written in Go, that allows a user to query a database in natural language. It connects you to DBpedia, a Sparql database that contains data from Wikipedia.

Its scope is still quite limited. The example questions are about all that is supported.

Please note that the results you are given may be incorrect, due to errors in the library, the data modelling, and the database itself (DBpedia).
